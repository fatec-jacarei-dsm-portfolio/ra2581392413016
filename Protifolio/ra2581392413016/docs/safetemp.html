PROJETO E MINHA PARTICIPAÇÃO

O SafeTemp foi desenvolvido como um aplicativo mobile voltado para o monitoramento em tempo real da temperatura de estufas do laboratorio. A solução foi criada em conjunto com um sensor físico projetado pelo próprio grupo, capaz de coletar dados continuamente e transmiti-los diretamente para o app via integração sem fio. O objetivo era facilitar o controle e vigia das estufas, permitindo ajustes rápidos e decisões mais precisas para garantir condições ideais de uso.

O projeto envolveu desde a concepção do hardware até a criação da interface digital, resultando em um sistema completo e acessível para acompanhamento das variações térmicas. Foram realizadas fases práticas de montagem, calibração do sensor, testes na estufa e validação do comportamento das leituras. Além disso, o time documentou cuidadosamente a identidade visual e estruturou o mapa da fase de descoberta, consolidando funcionalidades essenciais e comportamentos esperados do usuário, antes de realizar o upload final para o GitHub.

Minha participação no SafeTemp teve foco tanto na área de UX quanto na organização funcional do projeto. Fui responsável pelo mapeamento de usabilidade utilizando a técnica MoSCoW, definindo quais recursos eram indispensáveis (Must), desejáveis (Should), opcionais (Could) e descartáveis (Won’t) para a primeira versão do aplicativo. Esse processo garantiu que o desenvolvimento seguisse prioridades claras, equilibrando viabilidade técnica e valor para o usuário.

Também contribuí com a revisão das telas do frontend, verificando coerência visual, clareza das informações e fluidez da navegação. Meu trabalho ajudou na construção de uma experiência mais intuitiva, garantindo que os dados enviados pelo sensor fossem exibidos de forma simples e compreensível pelo usuário final.
